# Velours Fashion — Luxury Pakistani Women's Wear

Premium luxury e-commerce website for Pakistani women's fashion.

---

## 🚀 Netlify Deployment (Drag & Drop)

1. Go to https://app.netlify.com
2. Click **"Add new site" → "Deploy manually"**
3. Drag and drop the entire `velours-fashion` folder
4. Your site goes live instantly — no build command needed ✓

## ✨ Features

### Storefront
- Luxury hero section with animated background
- Product grid with filtering by category
- Sort by price / rating / newest
- Live search across all products
- Product quick view modal
- Size selection + quantity control
- Add to cart with localStorage persistence
- Animated cart sidebar
- Toast notifications

### Checkout
- Full checkout form (name, phone, email, address, city)
- 3 payment methods: COD, Bank Transfer, EasyPaisa/JazzCash
- Order confirmation with unique Order ID
- Orders saved to localStorage

### Admin Panel (Hidden)
- **Access:** Click the small "." in the footer 3 times
- **Username:** `admin`
- **Password:** `velours2025`
- Add new products (name, price, category, image URL, sizes, badge)
- Delete existing products
- View all orders with full details
